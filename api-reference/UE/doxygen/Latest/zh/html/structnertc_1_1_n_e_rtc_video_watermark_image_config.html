详细描述

图片水印设置参数。
最多可以添加 4 个图片水印。

类成员变量说明

◆ fps

int nertc::NERtcVideoWatermarkImageConfig::fps

播放帧率。默认 0 帧/秒，即不自动切换图片，图片单帧静态显示。注意：Windows端帧率不超过 20 fps。

◆ image_paths

char* nertc::NERtcVideoWatermarkImageConfig::image_paths[10]

水印图片路径。空时无效。

◆ loop

bool nertc::NERtcVideoWatermarkImageConfig::loop

是否设置循环。默认循环，设置为 false 后水印数组播放完毕后消失。

水印图片左上角与视频画布左上角的水平距离。单位为像素（pixel），默认值为 0。

◆ offset_y
