◆ bitrate

int com.netease.lava.nertc.sdk.video.NERtcEncodeConfig.bitrate = 0

The bitrate (Kbps) of video encoding. You can set the bitrate according to specific scenarios.

  • If the video bitrate is not reasonable,the SDK will adjust the bitrate.
  • If the bitrate is 0, the SDK will adjust accordingly.

◆ frameRate

NERtcVideoFrameRate com.netease.lava.nertc.sdk.video.NERtcEncodeConfig.frameRate = NERtcVideoFrameRate.FRAME_RATE_FPS_DEFAULT

The frame rate. For more information about frame rate of video encoding, see NERtcEncodeConfig.NERtcVideoFrameRate.

  • When using mainstreams, NERtcVideoFrameRate#FRAME_RATE_FPS_7 defines the frame rate by default。
  • NERtcConstants.VideoProfile >= NERtcConstants.VideoProfile::STANDARD，frameRate = FRAME_RATE_FPS_30 。
  • NERtcConstants.VideoProfile < NERtcConstants.VideoProfile::STANDARD，frameRate = FRAME_RATE_FPS_15 。
  • When using substreams, NERtcVideoFrameRate#FRAME_RATE_FPS_7 defines the frame rate by default.
