◆ loopCount

int com.netease.lava.nertc.sdk.audio.NERtcCreateAudioMixingOption.loopCount = 1

伴音循环播放的次数：

  • 1：（默认）播放一次。
  • ≤ 0：无限循环播放，直至调用 pauseAudioMixing 后暂停，或调用 stopAudioMixing 后停止。

◆ path

String com.netease.lava.nertc.sdk.audio.NERtcCreateAudioMixingOption.path

待播放的音乐文件的绝对路径或 URL 地址，支持本地 SD 卡中的绝对路径或 URL 地址。

  • 需精确到文件名及后缀，例如 “/sdcard/test.mp3”。
  • 支持的音效文件类型包括 MP3、M4A、AAC、3GP、WMA 和 WAV 格式。
