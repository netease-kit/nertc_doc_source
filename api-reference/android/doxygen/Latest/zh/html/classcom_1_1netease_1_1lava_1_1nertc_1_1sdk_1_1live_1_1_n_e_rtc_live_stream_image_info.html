◆ zOrder

int com.netease.lava.nertc.sdk.live.NERtcLiveStreamImageInfo.zOrder = 0

占位图的图层编号，用来决定渲染层级。
取值范围为 0~100，默认为 0。

  • 最小值为 0（默认值），表示该区域图像位于最底层。
  • 最大值为 100，表示该区域图像位于最顶层。
    注意：相同层级的渲染区域会按照数组中顺序进行渲染，随着 index 递增，依次往上叠加。
