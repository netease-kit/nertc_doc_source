◆ scaleMode

int com.netease.lava.nertc.sdk.video.NERtcVideoExternalBlendConfig.scaleMode

视频缩放模式。NERtcConstants.VideoScalingType.SCALE_ASPECT_FIT, NERtcConstants.VideoScalingType.SCALE_ASPECT_FILL, NERtcConstants.VideoScalingType.SCALE_ASPECT_BALANCED

  • NERtcConstants.VideoScalingType.SCALE_ASPECT_FIT: 适应视频，视频尺寸等比缩放。优先保证视频内容全部显示。若视频尺寸与显示视窗尺寸不一致，视窗未被填满的区域填充背景色。
  • NERtcConstants.VideoScalingType.SCALE_ASPECT_FILL: 适应区域，视频尺寸等比缩放。保证所有区域被填满，视频超出部分会被裁剪。
  • NERtcConstants.VideoScalingType.SCALE_ASPECT_BALANCED: 平衡模式，视频尺寸等比缩放。保证视频内容全部显示，视窗未被填满的区域填充背景色。
