resumeAudioMixing abstract method

Future<int> resumeAudioMixing()

该方法恢复混音，继续播放伴奏。

使用前提

已调用 startAudioMixing 方法开始播放音乐文件。

调用时机

请在引擎初始化之后调用此接口，该方法仅可在加入房间后调用。

相关接口

可以继续调用 pauseAudioMixing 方法恢复播放伴音文件。

返回值

  • 0（OK）：方法调用成功。
  • 其他：方法调用失败。
    • 30005（invalidState)：状态错误，比如引擎尚未初始化。
