设置当前伴音文件的音调。

通过此接口可以实现当本地人声和播放的音乐文件混音时，仅调节音乐文件的音调。

调用时机

请在引擎初始化之后调用此接口，且该方法仅可在加入房间后调用。

说明

当前伴音任务结束后，此接口的设置会恢复至默认。

参数说明

pitch 当前伴音文件的音调。默认值为 0，即不调整音调，取值范围为 -12 ~ 12，按半音音阶调整。每相邻两个值的音高距离相差半音；取值的绝对值越大，音调升高或降低得越多。
