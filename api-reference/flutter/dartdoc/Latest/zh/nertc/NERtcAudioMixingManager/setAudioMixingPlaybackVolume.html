调节伴奏播放音量。 该方法调节混音里伴奏的播放音量大小。

调用时机

请在引擎初始化之后调用此接口，该方法仅可在加入房间后调用。

参数说明

volume 伴奏发送音量。取值范围为 0~200。默认 100，即原始文件音量

返回值

  • 0（OK）：方法调用成功。
  • 其他：方法调用失败。
