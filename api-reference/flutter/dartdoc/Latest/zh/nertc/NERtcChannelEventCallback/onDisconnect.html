与服务器连接中断，可能原因包括：网络连接失败、服务器关闭该房间、用户被踢出房间等。

在调用 joinChannel 加入房间成功后，如果和服务器失去连接，就会触发该回调。

reason 连接中断的原因。详细信息请查看 NERtcErrorCode。
