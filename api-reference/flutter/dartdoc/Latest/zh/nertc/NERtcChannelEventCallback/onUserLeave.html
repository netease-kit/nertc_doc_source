远端用户离开房间的事件回调。

远端用户离开房间或掉线（在 40 ~ 50 秒内本端用户未收到远端用户的任何数据包）后，SDK 会触发该回调。

使用前提

请在 NERtcEngine.create 初始化接口中注册回调监听。

参数说明

uid 离开房间的远端用户 ID。 reason 该远端用户离开的原因，更多请参考 NERtcErrorCode。 leaveExtraInfo 该远端用户离开的额外信息。
