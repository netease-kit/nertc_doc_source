width

width property

int width
getter/setter pair

视频编码分辨率，衡量编码质量，以宽x高表示。与maxProfile属性二选一。 width表示视频帧在横轴上的像素，即自定义宽。

  • 设置为负数时表示采用 maxProfile档位。
  • 如果需要自定义分辨率场景，则设置此属性，maxProfile属性失效。

自定义视频输入 width 和 height 无效，会自动根据 maxProfile缩放。
