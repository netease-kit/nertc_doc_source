◆ KNERtcKeyDisableOverrideSpeakerOnReceiver

◆ KNERtcKeyDisableSWAECOnHeadset

NERTC_EXTERN_API NSString* const KNERtcKeyDisableSWAECOnHeadset

BOOL - Specifies whether to use echo reduction when the system sets the headset.

The default value is NO. If the value is set to YES, the SDK does not use the echo reduction in headset mode. The setting affects the audio quality of the headset in some mobile models.
