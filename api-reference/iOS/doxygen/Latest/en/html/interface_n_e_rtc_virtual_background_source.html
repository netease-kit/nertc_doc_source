◆ color

- (NSUInteger) color
readwritenonatomicassign

The color of the custom background image.

The format is a hexadecimal integer defined by RGB, without the # sign, such as 0xFFB6C1 for light pink. The default value is 0xFFFFFF, which signifies white. The value range is [0x000000,0xFFFFFF]. If the value is invalid, the SDK replaces the original background image with a white background image.

Note
This parameter takes effect only when the type of the custom background image is kNERtcBackgroundColor.
