该方法已经废弃，请使用 INERtcEngineEx#setExternalVideoSource:streamType: 方法代替。注意，如果已经调了此方法，将会导致再 INERtcEngine#setExternalVideoSource:streamType: 方法失效。 开启或关闭外部视频源数据输入。

成员 [NERtcEngineDelegateEx-p onNERtcEngineAudioHasHowling]
该回调将在未来版本中废弃，请使用- (void)onNERtcEngineAudioHasHowling: (NERtcEngineDelegateEx-p)flag. 检测到啸叫回调。 当声源与扩音设备之间因距离过近时，可能会产生啸叫。NERTC SDK 支持啸叫检测，当检测到有啸叫信号产生的时候，自动触发该回调直至啸叫停止。App 应用层可以在收到啸叫回调时，提示用户静音麦克风，或直接静音麦克风。
