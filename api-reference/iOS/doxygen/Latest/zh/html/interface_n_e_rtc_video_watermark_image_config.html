详细描述

视频图片水印配置。

注解
  • 如果设置了水印框宽或高，图像将缩放至水印框的宽高尺寸。
  • 如果设置了水印框宽高，按图像原始尺寸展示；原始尺寸大于视频尺寸时，缩放至视频尺寸。
  • 出于性能的考虑，建议设置尺寸合适的图片。
