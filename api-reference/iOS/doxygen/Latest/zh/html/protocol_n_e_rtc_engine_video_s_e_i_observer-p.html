详细描述

注册 SEI 观测者。 注册观测者并拉流成功后，当远端流发送 SEI，本端会收到 onNERtcEngineRecvSEIMsg 回调。

◆ onNERtcEngineRecvSEIMsg:message:

- (void) onNERtcEngineRecvSEIMsg: (uint64_t) userID
message: (NSData *) message 
optional

收到远端流的 SEI 内容回调。 当远端成功发送 SEI 后，本端会收到此回调。

注解
纯音频 SDK 禁用该回调，如需使用请前往云信官网下载并替换成视频 SDK。
参数
userID发送 SEI 的用户 ID。
message对应用户的 SEI 信息。
