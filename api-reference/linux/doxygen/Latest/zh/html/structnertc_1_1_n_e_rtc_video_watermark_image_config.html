◆ fps

unsigned int nertc::NERtcVideoWatermarkImageConfig::fps

播放帧率。默认值为 0，表示不自动切换图片，图片单帧静态展示。
帧率最高不超过 30fps，如果设置帧率高于视频流帧率，则按照视频流帧率展示。
