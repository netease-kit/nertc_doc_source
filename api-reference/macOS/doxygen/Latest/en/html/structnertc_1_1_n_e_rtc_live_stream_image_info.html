int nertc::NERtcLiveStreamImageInfo::x

The X parameter is used to set the horizontal coordinate value of the canvas. You can specify a point in the canvas with X and Y coordinates. This point is used as the anchor of the upper left corner of the placeholder image. Value range: 0 to 1920. If the specified value is set to an odd value, the value is automatically rounded down to an even number.

The Y parameter is used to set the vertical coordinate value of the canvas.
