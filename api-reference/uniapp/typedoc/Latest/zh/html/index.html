NERTC V5.6.38

NERTC Uniapp SDK

网易云信 NERTC SDK 提供完善的音视频通话开发框架，提供基于网络的点对点视频通话和语音通话功能，还提供多人视频和音频会议功能，支持通话中音视频设备控制和实时音视频模式切换，美颜等功能。

  • NERTC 是基础对象，包含应用程序调用的所有方法。

注意：错误码与安卓端相同。
