Interface NERtcAudioVolumeInfo

声音音量信息。 一个数组，包含每个说话者的用户 ID 和音量信息。

Hierarchy

  • NERtcAudioVolumeInfo

Index

Properties

  • subStreamVolume

  • userStringID

Properties

subStreamVolume

subStreamVolume: number

远端用户音频辅流的音量，取值范围为 [0,100] 。如果 subStreamVolume 为 0，表示该用户未发布音频辅流或音频辅流没有音量。

userID

userID: number

用户 ID，指定是哪个用户的音量。

userStringID

userStringID: number

用户字符串格式的 ID。

volume

volume: number

用户的音量，取值范围为 [0,100]。如果用户调用了 startAudioMixing，则 volume 为用户混音后的音量。如果 volume 为 0，表示该用户没有说话。如果数组为空，则表示此时远端没有人说话。
