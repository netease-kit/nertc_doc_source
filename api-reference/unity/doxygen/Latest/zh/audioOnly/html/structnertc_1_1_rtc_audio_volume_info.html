详细描述

声音音量信息。一个数组，包含每个说话者的用户 ID 和音量信息。

类成员变量说明

◆ uid

ulong nertc.RtcAudioVolumeInfo.uid

说话者的用户 ID。如果返回的 uid 为 0，则默认为本地用户。

◆ volume

uint nertc.RtcAudioVolumeInfo.volume

说话者的音量，范围为 0（最低）- 100（最高）。

◆ substreamVolume

uint nertc.RtcAudioVolumeInfo.substreamVolume

远端用户音频辅流的音量，取值范围为 [0,100] 。
如果 sub_stream_volume 为 0，表示该用户未发布音频辅流或音频辅流没有音量。
