详细描述

图片水印设置参数。
最多可以添加 4 个图片水印。

类成员变量说明

◆ imagePaths

string [] nertc.RtcImageWatermarkConfig.imagePaths

水印图片路径。空时无效。

◆ wmAlpha

int nertc.RtcImageWatermarkConfig.wmAlpha

水印框内背景颜色透明度。

◆ wmWidth

int nertc.RtcImageWatermarkConfig.wmWidth

水印框的宽度。单位为像素（pixel），默认值为 0，表示没有水印框。

◆ wmHeight

int nertc.RtcImageWatermarkConfig.wmHeight

水印框的高度。单位为像素（pixel），默认值为 0，表示没有水印框。

◆ offsetX

int nertc.RtcImageWatermarkConfig.offsetX

水印图片左上角与视频画布左上角的水平距离。单位为像素（pixel），默认值为 0。

◆ offsetY

int nertc.RtcImageWatermarkConfig.offsetY

水印图片左上角与视频画布左上角的垂直距离。单位为像素（pixel），默认值为 0。

◆ fps

int nertc.RtcImageWatermarkConfig.fps

播放帧率。默认 0 帧/秒，即不自动切换图片，图片单帧静态显示。注意：Windows 端帧率不超过 20 fps。

◆ loop

bool nertc.RtcImageWatermarkConfig.loop

是否设置循环。默认循环，设置为 false 后水印数组播放完毕后消失。
