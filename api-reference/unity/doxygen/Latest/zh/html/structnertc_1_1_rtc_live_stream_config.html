◆ singleVideoPassthrough

bool nertc.RtcLiveStreamConfig.singleVideoPassthrough

单路视频透传开关，默认为关闭状态。

  • 开启后，如果房间中只有一路视频流输入， 则不对输入视频流进行转码，不遵循转码布局，直接推流 CDN。
  • 如果有多个房间成员视频流混合为一路流，则该设置失效，并在恢复为一个成员画面（单路流）时也不会恢复。

◆ audioBitrate

int nertc.RtcLiveStreamConfig.audioBitrate

音频推流码率。

  • 单位为 kbps，取值范围为 10~192。
  • 语音场景建议设置为 64 及以上码率，音乐场景建议设置为 128 及以上码率。

◆ sampleRate

RtcLiveStreamAudioSampleRate nertc.RtcLiveStreamConfig.sampleRate

音频推流采样率。单位为Hz。默认为 kNERtcLiveStreamAudioSampleRate48000，即采样率为 48 kHz。

◆ channels

int nertc.RtcLiveStreamConfig.channels

音频推流声道数，默认值为 2 双声道。

◆ audioCodecProfile

RtcLiveStreamAudioCodecProfile nertc.RtcLiveStreamConfig.audioCodecProfile

音频编码规格。默认值 kNERtcLiveStreamAudioCodecProfileLCAAC，普通编码规格。

  • 0: LC-AAC 规格，表示基本音频编码规格
  • 1: HE-AAC 规格，表示高效音频编码规格。
