x 参数用于设置画布的横轴坐标值。 通过 x 和 y 指定画布坐标中的一个点，该点将作为占位图片的左上角。 取值范围为 0~1920，若设置为奇数值，会自动向下取偶。

  • 通过 x 和 y 指定画布坐标中的一个点，该点将作为占位图片的左上角。
  • 取值范围为 0~1920，若设置为奇数值，会自动向下取偶。

◆ zOrder

int nertc.RtcLiveStreamImageInfo.zOrder

占位图的图层编号，用来决定渲染层级。
取值范围为 0~100，默认为 0。

  • 最小值为 0（默认值），表示该区域图像位于最底层。
  • 最大值为 100，表示该区域图像位于最顶层。
    注意：相同层级的渲染区域会按照数组中顺序进行渲染，随着 index 递增，依次往上叠加。
