Interface NERtcEncoderWatermarkConfig

编码水印设置。

同时设置文字、时间戳或图片水印时，如果不同类型的水印位置有重叠，会按照图片、文本、时间戳的顺序进行图层覆盖。

mediaType

视频流类型。支持设置为主流（video）或辅流（screen）。

时间戳水印。只能添加 1 个时间戳水印。
