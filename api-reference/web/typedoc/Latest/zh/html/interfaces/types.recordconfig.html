Properties

isHostSpeaker

isHostSpeaker: boolean

当前用户是否是主讲人。

recordAudio

recordAudio: boolean

是否开启音频实时音录制，false 不需要，true 需要。默认 false。

recordType

recordType: 0 | 1 | 2

录制模式。

  • 0：合流录制 + 单流录制。
  • 1：合流录制模式。只产生混合录制文件。
  • 2：单流录制模式。只产生单独录制文件。

recordVideo

recordVideo: boolean

是否开启视频实时音录制，false 不需要，true 需要。默认 false。
